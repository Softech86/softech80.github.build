# A Blog of Softech86

#博客

Softech86 的个人博客：收集markdown及资源文件，生成对应的静态博客。

**Version 1.0**

## 安装
需要  `node` 支持。

### 安装 node_modules
```
$ npm install # or yarn install
```

### 编译 markdown 文件夹
```
$ gulp
```

### 生成静态文件
```
$ npm run generate
```

### 更多支持
For detailed explanation on how things work, checkout the [Nuxt.js docs](https://github.com/nuxt/nuxt.js).

## 技术栈
- Vue
- Nuxt
- Gulp
